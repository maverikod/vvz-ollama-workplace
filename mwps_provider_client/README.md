# mwps_provider_client

**Author:** Vasiliy Zdanovskiy  
**email:** vasilyvz@gmail.com

Provider client for **Model Workplace Server**: implements root [provider_client_standard](../../docs/standards/provider_client_standard.md). For the model workspace, **Model Workplace Server server is just a separate provider**; this package is the client that talks to mwps-adapter (or directly to Model Workplace Server). PyPI-publishable.

**Structure:** `docs/` (ТЗ, reports/, plans/), `src/mwps_provider_client/`, `tests/` (unit/, integration/), `config/`, `examples/`, `code_analysis/` (generated), `projectid`, `pyproject.toml`, `.gitignore`.
